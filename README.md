## Description

A simple weather API developed in NestJS

## Features

- Weather Endpoint
- Forecast Endpoint
- Geolocation Endpoints
- DTOs Validation/Transformation
- Rate Limiting
- Caching Responses (Redis)
- Swagger Documentation (/docs)
- Unit Testing
- E2E Testing

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

## Documentation

`http//localhost:8080/docs`

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
